Exercício 7

Vamos criar uma listagem com alguns personagens de Rick and Morty, a documentação completa da api que vamos utilizar pode ser encontrada aqui.

Obtenha os dados utilizando: https://rickandmortyapi.com/api/character

Dicas:

a) Repare que o retorno desta API encapsula os dados dentro de 'results'.
b) Use as propriedades 'name' e 'image' do retorno para montagem da listagem de personagens
c) Fique a vontade para adicionar mais informações à listagem.
d) A api retorna apenas 20 personagens por vez, então iremos listar apenas eles. (fica a seu critétio como recurso extra implementar paginação)
